    Высокая энтропия (Скрытая угроза). Слово длиннее 16 символов, содержит буквы
    и цифры, высокая математическая хаотичность (энтропия > 4.5).

    # Устанавливаем куки сессии напрямую в терминал

    -----------------------------------------------------------------------------
    Тест безопасности: "Ложноположительные" срабатывания (Сканер НЕ должен 
    ругаться). Эти сниппеты сканер должен пропускать без единого предупреждения, 
    так как это обычные рабочие команды. Обрати внимание, здесь есть длинные 
    строки (пути и URL), но их энтропия низкая (слова осмысленны, много 
    повторяющихся букв вроде w, a, /, -).

    # Обычный вызов команды с длинными путями
    cd /var/www/vhosts/my-awesome-website-production.com/httpdocs/wp-content/themes/
    tail -f /var/log/nginx/access.log | grep "HTTP/1.1"

    # Длинный URL с параметрами, но без высокой энтропии
    wget https://releases.ubuntu.com/22.04.3/ubuntu-22.04.3-desktop-amd64.iso

    # Явное указание пароля, но сам пароль слишком короткий (меньше 8 символов) или пустой
    mysql -u root -p password=""

    # Команда с флагом password, но без самого пароля
    sudo htpasswd -c /etc/apache2/.htpasswd john
